1. System Overview
This document is the complete technical and operational blueprint for building an agentic lead generation system tailored to a Pakistani software development agency seeking foreign clients. The system uses LangChain and LangGraph to orchestrate a multi-agent pipeline powered by OpenAI GPT-4o, integrated with Gmail and Google Sheets for outreach and tracking.
The system runs daily, finds qualified leads in US, UK, and UAE markets, enriches them with contact details and context, generates hyper-personalized outreach emails, sends them via Gmail, manages follow-up sequences, and logs everything to Google Sheets — all autonomously.
What This System Replaces
• 3–5 hours/day of manual prospecting on LinkedIn, Crunchbase, Upwork
• Hand-written cold emails that are generic and get ignored
• Missed follow-ups because you forgot or got busy with client work
• No CRM — no visibility into what's working or where leads stand
• Inconsistent outreach — sometimes daily, sometimes nothing for a week
What This System Delivers
• 20–30 qualified leads discovered and scored automatically every morning
• Unique, personalized email written per lead — not a template
• Automated 3-touch email sequence (Day 1 → Day 4 → Day 9)
• Real-time CRM in Google Sheets — every lead tracked, every reply flagged
• 1–2 discovery calls booked per week within the first 30 days

3. The 6-Node LangGraph Pipeline
The system is built as a LangGraph StateGraph with 6 nodes. Each node is a LangChain agent or tool chain. They run sequentially every morning, triggered by a scheduler.

START

  ▼
[ node_1_discover ]  ──  Crunchbase + LinkedIn + IndieHackers + PH
  │
  ▼
[ node_2_score ]     ──  GPT-4o scores each lead against ICP (1–10)
  │  filter: score >= 7
  ▼
[ node_3_enrich ]    ──  Hunter.io + Apollo + LinkedIn scrape
  │
  ▼

  ▼
END  (monitor loop checks replies every 2 hours)

6. LangGraph State Definition
The AgentState is passed through every node. Each node reads from it and writes back to it. This is the core data contract of the pipeline.
# graph/state.py
from typing import TypedDict, List, Optional, Dict, Any
from langgraph.graph import StateGraph, END
class LeadState(TypedDict):
    # Raw lead from discovery
    company_name:  str
    website:       str
    linkedin_url:  str
    location:      str
    team_size:     int
    funding:       Optional[Dict]
    source:        str
    raw_signals:   List[str]
    # After scoring (Node 2)
    icp_score:     Optional[int]
    score_reason:  Optional[str]
    hot_signals:   Optional[List[str]]
    # After enrichment (Node 3)
    contact:       Optional[Dict]
    tech_stack:    Optional[List[str]]
    hook:          Optional[str]
    # After personalization (Node 4)
    email_subject: Optional[str]
    email_body:    Optional[str]
    # After outreach (Node 5)
    email_sent:    Optional[bool]
    email_sent_at: Optional[str]
    # After CRM (Node 6)
    crm_row_id:    Optional[str]
    status:        Optional[str]   # 'sent', 'replied', 'booked', 'skip'
class PipelineState(TypedDict):
    leads:         List[LeadState]
    qualified:     List[LeadState]  # score >= 7
    processed:     List[LeadState]  # completed pipeline
    run_date:      str
    errors:        List[str]
# graph/pipeline.py
from langgraph.graph import StateGraph, END
from agents.discovery_agent     import run_discovery
from agents.scoring_agent       import run_scoring
from agents.enrichment_agent    import run_enrichment
from agents.personalization_agent import run_personalization
from agents.outreach_agent      import run_outreach
from agents.crm_agent           import run_crm
def build_pipeline() -> StateGraph:
    graph = StateGraph(PipelineState)
    graph.add_node('discover',     run_discovery)
    graph.add_node('score',        run_scoring)
    graph.add_node('enrich',       run_enrichment)
    graph.add_node('personalize',  run_personalization)
    graph.add_node('outreach',     run_outreach)
    graph.add_node('crm',          run_crm)
    graph.set_entry_point('discover')
    graph.add_edge('discover',    'score')
    graph.add_edge('score',       'enrich')      # filtered inside node
    graph.add_edge('enrich',      'personalize')
    graph.add_edge('personalize', 'outreach')
    graph.add_edge('outreach',    'crm')
    graph.add_edge('crm',          END)
    return graph.compile()

7.2 Personalization System Prompt
prompts/personalization_prompt.py — PERSONALIZATION_SYSTEM_PROMPT
You are a senior business development manager at a top-tier software agency.
You write cold emails that get replies. You write like a human, not a marketer.
Rules (follow strictly):
1. Subject line: Max 10 words. Reference something SPECIFIC about them (funding, product, post).
2. Opening: First sentence must mention ONE specific thing about them. No generic openers.
3. Pain bridge: Connect their situation to what we solve in 1–2 sentences.
4. Social proof: Mention similar work we've done. Be specific about industry/feature.
5. CTA: ONE low-friction ask only. 'Worth a 20-min call?' or 'Want me to send a case study?'
6. Total length: 80–120 words max.
7. Tone: Peer-to-peer, confident, never desperate or salesy.
8. Never mention Pakistan in email 1. Let the quality speak first.
9. Never say 'I hope this finds you well' or any generic opener.
10. Never use bullet points or headers in the email body.
Respond ONLY in JSON (no markdown):
{ 'subject': '<subject line>', 'body': '<email body>' }

# scheduler/run_pipeline.py (Phase 1 — MVP)
import os, json
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import JsonOutputParser
llm = ChatOpenAI(model='gpt-4o', api_key=os.getenv('OPENAI_API_KEY'))
def score_lead(lead_info: str) -> dict:
    chain = ChatPromptTemplate.from_messages([
        ('system', SCORING_SYSTEM_PROMPT),
        ('human', 'Score this lead: {lead}')
    ]) | llm | JsonOutputParser()
    return chain.invoke({'lead': lead_info})
def write_email(lead_context: str) -> dict:
    chain = ChatPromptTemplate.from_messages([
        ('system', PERSONALIZATION_SYSTEM_PROMPT),
        ('human', '{context}')
    ]) | llm | JsonOutputParser()
    return chain.invoke({'context': lead_context})
if __name__ == '__main__':
    lead_info = input('Paste company info: ')
    score = score_lead(lead_info)
    print(f'Score: {score["score"]} — {score["reason"]}')
    if score['score'] >= 7:
        email = write_email(lead_info)
        print(f'Subject: {email["subject"]}')
        print(f'Body:\n{email["body"]}')
